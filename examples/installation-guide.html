🎤 AI Assistant 원클릭 설치

💾 방법 1: 자동 시작 설치

<script src="https://cdn.ai-assistant.com/v1/standalone.js" data-auto-start></script>

data-auto-start 속성으로 페이지 로드 시 자동 실행

⚙️ 방법 2: 수동 설정

<script src="https://cdn.ai-assistant.com/v1/standalone.js"></script> <script> AIAssistant.init({ language: 'ko-KR', voiceModel: 'nova', enableLogging: true }) </script>

🔑 방법 3: API 키 사용 (고품질 TTS)

<script> AIAssistant.init({ apiKey: 'your-api-key', voiceModel: 'nova' // 클라우드 Nova 음성 사용 }) </script>

🎯 테스트 요소들


🗣️ 사용 가능한 음성 명령어

  • "로그인 클릭해줘" - 로그인 버튼 클릭
  • "검색해줘" - 검색창 포커스
  • "아니" - 이전 액션 취소
  • "확인, 아니 취소" - 복합 명령 처리
  • "위로" - 페이지 상단 스크롤
  • "아래로" - 페이지 하단 스크롤
