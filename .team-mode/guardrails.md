# Team Mode 가드레일 (상세)

| 프로필 | 할 수 있는 것 | 할 수 없는 것 |
| --- | --- | --- |
| **Maker (개발)** | 코드·스키마·인프라 설정, 문서, PR 생성/머지(리뷰 후) | 비밀값 커밋, `main` 직접 push |
| **Planner (기획·디자인·QA·CS)** | `docs/`·`design-system` 문서·명세, 이슈·리뷰 코멘트 | 코드 레벨 수정(디자인시스템 제외) |
| **Lead (PMO/PL)** | 위 전체 + 프로필·CODEOWNERS·릴리스·의사결정·자율등급 승격 승인 | 비밀값 커밋 |

## 에이전트에 대한 추가 규칙
- 접근권한(GitHub) ≠ 행동경계(Team Mode). 둘 다 만족해야 변경 성립.
- **어떤 에이전트도 자기 권한/자율등급/합격점을 스스로 높일 수 없다.** 상향은 사람 승인 + ADR.
- 새 에이전트는 **Foundry 심의**를 거쳐야만 태어난다. 태어난 직후는 `probation`(자율 한 단계 하향).
- 위반 시: PR 자동 차단 + Lead 검토. 예외는 ADR로 기록.
